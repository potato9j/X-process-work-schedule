# 통합 READ.ME

## [CCTV근무, 불침번 공정코드]

- 프로젝트 개요 : 모든 공정한 방법으로 근무표를 작성해도 불평, 불만의 목소리를 접수받아왔음. 이에대해 근무표를 작성하는 피로도가 높았음. 이 문제의 순환고리를 끊어내기위해 개발자의 막대한 개인정비시간을 소요하여 코드를 작성하였음.

- 소요 기간 : 13시간
