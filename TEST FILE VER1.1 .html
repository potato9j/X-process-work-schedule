근무 공정표

근무는 총 4개 타임으로 06시~12시, 12시~18시, 18시~00시, 00시~06시 입니다.

각 근무 1개 타임별로 2명씩 짝지어서 두팀으로 한타임에 총 4명이 근무에 투입됩니다.

근무 공정표는 난수 이차배열을 이용하여 자동으로 생성됩니다.

날짜 06시~12시 12시~18시 18시~00시 00시~06시
